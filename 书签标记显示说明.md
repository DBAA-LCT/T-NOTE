# 书签标记显示说明

## 功能概述

在编辑器内容的左侧边缘显示书签标记，让用户可以直观地看到书签在文档中的具体位置。

## 显示效果

### 布局结构
```
┌─────────────────────────────────────┐
│ 工具栏                               │
├─────────────────────────────────────┤
│ ▌ 这是第一行内容...                 │
│   这是第二行内容...                 │
│ ▌ 这里有一个书签标记                │
│   继续编辑内容...                   │
└─────────────────────────────────────┘
```

## 视觉设计

### 书签标记样式
- **位置**：编辑器内容左侧边缘
- **形状**：竖条状标记
- **尺寸**：4px 宽 × 16px 高
- **颜色**：橙色渐变 (#fa8c16 → #faad14)
- **圆角**：2px，柔和的边缘
- **阴影**：轻微的橙色阴影

### 悬停效果
- 标记变宽：4px → 6px
- 标记变高：16px → 20px
- 阴影增强：更明显的光晕效果
- 显示书签名称的提示框

### 提示框样式
- **背景**：深灰色 (#262626)
- **文字**：白色
- **位置**：标记右侧
- **箭头**：指向标记的小三角
- **动画**：淡入淡出效果

## 交互功能

### 1. 视觉提示
- 书签位置一目了然
- 橙色标记醒目但不突兀
- 与编辑内容自然融合

### 2. 悬停交互
- 鼠标悬停时标记放大
- 显示书签名称
- 提供视觉反馈

### 3. 点击跳转
- 点击标记跳转到书签位置
- 光标自动定位
- 显示跳转成功提示

## 技术实现

### 位置计算
- 使用 Quill 的 `getBounds()` API 获取书签位置
- 标记与书签所在行垂直对齐
- 使用绝对定位固定在左侧

### DOM 结构
```html
<div class="bookmark-marker">
  <div class="bookmark-indicator"></div>
  <div class="bookmark-tooltip">书签名称</div>
</div>
```

### 布局调整
- 编辑器左侧增加 20px 内边距
- 为书签标记预留空间
- 确保标记不遮挡内容

## 优势特点

1. **直观可见**：
   - 在编辑时直接看到书签位置
   - 不需要打开书签列表

2. **不干扰编辑**：
   - 标记位于边缘，不影响阅读
   - 小巧的设计，视觉负担小

3. **快速定位**：
   - 一眼看出文档中有多少书签
   - 了解书签的分布情况

4. **交互友好**：
   - 悬停显示书签名称
   - 点击即可跳转
   - 平滑的动画效果

## 使用场景

- **长文档导航**：快速找到标记的重要段落
- **内容审阅**：标记需要修改的位置
- **协作编辑**：标记讨论点或待办事项
- **学习笔记**：标记重点内容

## 视觉对比

### 之前（右侧大图标）
- 占用空间大
- 需要额外的右侧边距
- 可能影响布局

### 现在（左侧小标记）
- 占用空间小（仅 4px）
- 与内容自然融合
- 更符合编辑器习惯
