# UI 功能说明

## 🎨 界面特性

### 侧边栏功能
- **文件操作区**
  - 打开按钮：打开已保存的笔记文件
  - 保存按钮：快速保存当前笔记（Ctrl+S）
  - 另存为按钮：将笔记保存到新位置

- **页面管理**
  - 新建页面：创建新的笔记页面
  - 页面列表：显示所有页面，点击切换
  - 删除页面：带确认提示的删除功能
  - 页面信息：显示标题、标签和更新时间

- **搜索功能**
  - 按标签搜索：输入标签名快速筛选页面
  - 实时过滤：输入即时显示结果

### 编辑器功能
- **标题编辑**
  - 大号输入框，支持实时编辑
  - 自动保存更改

- **标签管理**
  - 添加标签：输入后按回车或点击添加按钮
  - 删除标签：点击标签上的关闭图标
  - 彩色标签：蓝色主题，易于识别

- **富文本编辑**
  - 标题：支持 H1、H2、H3 三级标题
  - 文本样式：粗体、斜体、下划线、删除线
  - 列表：有序列表、无序列表
  - 颜色：文字颜色、背景颜色
  - 对齐：左对齐、居中、右对齐
  - 插入：链接、图片
  - 代码块：支持代码高亮
  - 清除格式：一键清除所有格式

## 🎯 设计亮点

1. **现代化设计**
   - 采用 Ant Design 6.0 设计系统
   - 统一的蓝色主题色 (#1677ff)
   - 8px 圆角设计，柔和美观

2. **交互优化**
   - 悬停效果：按钮和列表项有明显反馈
   - 选中状态：当前页面高亮显示
   - 确认对话框：删除操作需要确认
   - 消息提示：操作成功/失败有友好提示

3. **视觉层级**
   - 清晰的信息分组
   - 合理的间距和留白
   - 图标辅助理解功能

4. **细节打磨**
   - 自定义滚动条样式
   - 优化的富文本编辑器样式
   - 代码块特殊样式
   - 中文工具栏提示

## 🚀 快捷键

- **Ctrl + S**：保存笔记
- **Enter**：在标签输入框中添加标签

## 💡 使用建议

1. **组织笔记**
   - 使用标签对页面进行分类
   - 利用搜索功能快速定位
   - 定期保存避免数据丢失

2. **编辑技巧**
   - 使用标题组织内容结构
   - 利用列表整理要点
   - 代码块适合记录代码片段
   - 颜色标记重点内容

3. **文件管理**
   - 建议定期使用"另存为"备份
   - 笔记文件为 JSON 格式
   - 可以在不同设备间复制文件
