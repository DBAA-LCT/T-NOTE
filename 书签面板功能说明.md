# 书签面板功能说明

## 功能概述

在侧边栏添加了专门的书签面板，使用二级折叠列表展示所有页面的书签。

## 访问方式

点击左侧图标栏的第四个图标 📖（书签图标），即可打开书签面板。

## 界面设计

### 二级列表结构
- **一级列表**：页面列表（可折叠）
- **二级列表**：每个页面的书签列表

### 默认展开规则
- 默认只展开当前正在编辑的页面
- 其他页面默认折叠
- 可以手动点击展开/折叠任意页面

## 功能特性

### 1. 页面列表（一级）
每个页面项显示：
- **展开/折叠图标**：右箭头，展开时旋转 90 度
- **页面标题**：页面名称
- **书签数量**：该页面的书签总数
- **高亮显示**：当前页面用蓝色边框标识

### 2. 书签列表（二级）
展开页面后显示该页面的所有书签：
- **书签图标**：橙色 📖
- **书签名称**：书签的名称
- **书签位置**：position 索引值
- **删除按钮**：红色删除图标
- **悬停效果**：鼠标悬停时背景变灰

### 3. 快速跳转
- **点击书签**：自动切换到对应页面并跳转到书签位置
- **跨页面跳转**：如果书签在其他页面，会先切换页面再跳转
- **同页面跳转**：如果书签在当前页面，直接跳转

### 4. 删除书签
- 每个书签右侧有删除按钮
- 点击删除按钮会弹出确认对话框
- 确认后删除该书签

## 视觉设计

### 折叠面板
- **背景**：白色
- **边框**：灰色（普通页面）/ 蓝色（当前页面）
- **圆角**：8px
- **间距**：8px
- **展开图标**：右箭头，带旋转动画

### 页面标题
- **字体**：加粗
- **颜色**：黑色（普通）/ 蓝色（当前页面）
- **书签数量**：灰色小字，显示在右侧

### 书签项
- **图标**：橙色书签图标
- **字体**：13px
- **悬停**：背景变为浅灰色
- **删除按钮**：红色危险按钮

### 空状态
- 显示大号书签图标
- 提示"暂无书签"

## 交互细节

### 折叠/展开
- 点击页面标题区域可以折叠/展开
- 展开图标会旋转 90 度
- 平滑的展开/折叠动画

### 悬停反馈
- 书签项悬停时背景变灰
- 提供清晰的交互反馈

### 当前页面标识
- 蓝色边框
- 蓝色标题文字
- 默认展开状态

## 使用场景

1. **书签总览**：
   - 查看所有页面的书签分布
   - 了解哪些页面有书签及数量

2. **快速导航**：
   - 从书签面板直接跳转到任意书签
   - 无需先找到页面再找书签

3. **分页管理**：
   - 按页面组织书签
   - 折叠不需要的页面，保持界面整洁

4. **跨页面查找**：
   - 在多个页面中查找特定书签
   - 快速定位重要内容

## 优势特点

1. **层次清晰**：
   - 二级结构，组织有序
   - 页面和书签分层展示

2. **节省空间**：
   - 默认只展开当前页面
   - 折叠不需要的内容

3. **操作便捷**：
   - 一键展开/折叠
   - 点击即可跳转

4. **视觉友好**：
   - 当前页面高亮
   - 清晰的视觉层次
