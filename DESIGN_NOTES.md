# 设计说明

## 🎨 设计理念

这次 UI 升级遵循以下设计原则：

### 1. 简洁明了
- 去除不必要的装饰元素
- 突出核心功能
- 清晰的视觉层级

### 2. 专业美观
- 使用 Ant Design 企业级设计系统
- 统一的配色方案
- 精心设计的间距和排版

### 3. 易用性优先
- 图标 + 文字，降低学习成本
- 合理的按钮分组
- 明确的操作反馈

## 🎨 配色方案

### 主色调
- **主题色**：#1677ff（Ant Design 蓝）
- **成功色**：#52c41a
- **警告色**：#faad14
- **危险色**：#ff4d4f

### 中性色
- **背景色**：#ffffff（白色）
- **次级背景**：#fafafa（浅灰）
- **边框色**：#f0f0f0（灰色）
- **文字色**：#000000（黑色）
- **次级文字**：#8c8c8c（灰色）

### 状态色
- **选中状态**：#e6f4ff（浅蓝背景）+ #1677ff（蓝色边框）
- **悬停状态**：轻微的阴影和背景色变化
- **禁用状态**：降低透明度

## 📐 尺寸规范

### 间距
- **小间距**：8px
- **中间距**：16px
- **大间距**：24px
- **超大间距**：32px

### 圆角
- **统一圆角**：8px
- **标签圆角**：4px

### 字体
- **标题**：24px / 600
- **正文**：16px / 400
- **小字**：12px / 400

## 🔧 组件使用

### Ant Design 组件清单
- Layout（布局）
  - Sider（侧边栏）
  - Content（内容区）
- Button（按钮）
- Input（输入框）
- Search（搜索框）
- List（列表）
- Tag（标签）
- Space（间距）
- Typography（排版）
- Divider（分割线）
- Empty（空状态）
- Popconfirm（确认框）
- message（消息提示）
- ConfigProvider（全局配置）

### 图标使用
- PlusOutlined（新增）
- SaveOutlined（保存）
- SaveFilled（另存为）
- FolderOpenOutlined（打开）
- DeleteOutlined（删除）
- FileTextOutlined（文件）
- SearchOutlined（搜索）
- CloseOutlined（关闭）

## 🌟 特色功能

### 1. 智能列表
- 显示页面标题、标签、更新时间
- 当前页面高亮显示
- 悬停效果提升交互体验
- 删除按钮带确认对话框

### 2. 标签系统
- 彩色标签，易于识别
- 支持添加和删除
- 可按标签搜索过滤

### 3. 富文本编辑
- 完整的格式化工具栏
- 中文工具提示
- 优化的编辑器样式
- 代码块特殊样式

### 4. 操作反馈
- 保存成功提示
- 删除确认对话框
- 空状态友好提示

## 🚀 性能优化

1. **按需加载**：Ant Design 支持 Tree Shaking
2. **样式隔离**：使用 CSS-in-JS 避免样式冲突
3. **组件复用**：统一的组件使用方式
4. **响应式设计**：自适应不同窗口大小

- 适配 Windows 窗口控件
- 优化滚动条样式
- 支持键盘快捷键
- 原生文件对话框集成
