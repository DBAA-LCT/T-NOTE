# 书签显示功能说明

## 功能概述

在页面列表中显示书签信息，让用户可以快速了解哪些页面包含书签以及书签的内容。

## 显示位置

### 1. 页面面板 (PagesPanel)
- **书签数量徽章**：在页面标题右侧显示橙色书签图标和数量
- **书签列表预览**：显示前 3 个书签的名称
- **更多提示**：如果超过 3 个书签，显示 "+N" 表示还有更多

### 2. 搜索面板 (SearchPanel)
- **书签数量徽章**：在页面标题后显示书签数量
- **书签列表预览**：显示前 2 个书签的名称
- **更多提示**：如果超过 2 个书签，显示 "+N" 表示还有更多

## 视觉设计

### 颜色方案
- **书签标签**：橙色 (color="orange")，与蓝色的标签区分
- **书签图标**：📖 BookOutlined 图标
- **字体大小**：11-12px，保持简洁

### 布局结构
```
┌─────────────────────────────────┐
│ 页面标题              [📖 3] [×] │
│ [标签1] [标签2]                 │
│ [📖 书签1] [📖 书签2] [📖 书签3] │
│ 2024-02-06 15:30:00            │
└─────────────────────────────────┘
```

## 交互提示

- **悬停提示**：鼠标悬停在书签数量徽章上，显示 "N 个书签"
- **点击页面**：点击页面项可以切换到该页面进行编辑
- **书签管理**：在编辑器工具栏中点击书签按钮进行详细管理

## 优势

1. **快速识别**：一眼就能看出哪些页面有书签
2. **内容预览**：直接看到书签名称，了解标记的内容
3. **视觉区分**：橙色书签与蓝色标签形成对比，易于区分
4. **信息密度**：在有限空间内展示关键信息

## 使用场景

- 快速找到包含重要标记的页面
- 了解页面的书签分布情况
- 通过书签名称预览页面内容重点
