# 书签功能说明

## 功能概述

为编辑器添加了书签功能，可以在文档中标记重要位置并快速跳转。

## 使用方法

### 1. 添加书签
1. 在编辑器中将光标移动到想要添加书签的位置
2. 点击右上角的"书签"按钮
3. 在弹出的面板中输入书签名称
4. 点击"添加"按钮或按回车键

**提示：** 如果没有选中文本，书签会添加到当前光标位置

### 2. 查看书签
- 点击"书签"按钮查看当前页面的所有书签
- 按钮上会显示书签数量，如：书签 (3)

### 3. 跳转到书签
- 在书签列表中点击任意书签
- 编辑器会自动滚动并定位到该书签位置
- 光标会移动到书签所在位置

### 4. 删除书签
- 在书签列表中点击书签右侧的删除图标
- 确认删除操作

## 技术实现

### 数据结构
```typescript
interface Bookmark {
  id: string;           // 唯一标识
  name: string;         // 书签名称
  position: number;     // Quill 编辑器中的位置索引
  createdAt: number;    // 创建时间
}
```

### 核心功能
- **位置记录**：使用 Quill 编辑器的 position index 记录书签位置
- **精确跳转**：通过 `setSelection()` 和 `scrollIntoView()` 实现精确定位
- **持久化**：书签数据保存在页面对象中，随笔记一起保存

## 使用场景

1. **长文档导航**：在长文档中标记章节或重要段落
2. **待办标记**：标记需要后续编辑的位置
3. **参考点**：标记需要频繁查看的内容
4. **代码片段**：在技术笔记中标记重要代码位置

## 界面位置

书签按钮位于编辑器顶部区域，标签输入框的右侧，方便快速访问。
