# 侧边栏功能说明

## 新增功能

已为项目添加了侧边图标栏，包含以下功能：

### 1. 图标栏 (IconBar)
- 位置：最左侧，深色背景
- 宽度：50px
- 包含三个图标：
  - 📄 **页面** - 管理所有笔记页面
  - 🔍 **搜索** - 按标签搜索页面
  - ☑️ **TODO** - 待办事项（功能预留）

### 2. 交互方式
- 点击图标：显示对应的二级菜单面板
- 再次点击：关闭当前面板
- 激活状态：图标显示蓝色高亮

### 3. 二级菜单面板

#### 页面面板 (PagesPanel)
- 显示所有笔记页面列表
- 支持新建页面
- 支持删除页面
- 点击页面可切换编辑

#### 搜索面板 (SearchPanel)
- 按标签搜索页面
- 实时过滤显示结果
- 点击搜索结果可跳转到对应页面

#### TODO 面板 (TodoPanel)
- 当前为占位界面
- 预留待办事项功能

### 4. 顶部工具栏 (TopBar)
- 显示笔记名称
- 提供打开、保存、另存为按钮
- 从侧边栏移至顶部，更符合常规应用布局

## 文件结构

```
src/components/
├── IconBar.tsx          # 图标栏组件
├── IconBar.css          # 图标栏样式
├── PagesPanel.tsx       # 页面管理面板
├── SearchPanel.tsx      # 搜索面板
├── TodoPanel.tsx        # TODO 面板
└── TopBar.tsx           # 顶部工具栏
```
