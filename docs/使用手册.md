## 快速入门

### 第一次使用

1. **启动应用**：双击桌面上的 T-Note 图标
2. **创建笔记**：点击顶部菜单"文件" → "新建笔记"，选择保存位置
3. **创建页面**：点击左侧"新建页面"按钮
4. **开始编辑**：在编辑器中输入内容
5. **保存**：按 Ctrl+S 保存

### 打开已有笔记

- **方法一**：点击"文件" → "打开笔记"，选择 .note 文件
- **方法二**：直接双击 .note 文件
- **方法三**：从"最近打开"列表中选择

## 笔记管理

### 创建新笔记

1. 点击"文件" → "新建笔记"
2. 选择保存位置和文件名
3. 开始创建页面和编辑内容

### 打开笔记

**方法一：通过菜单**
1. 点击"文件" → "打开笔记"
2. 选择 .note 文件

**方法二：双击文件**
- 直接双击 .note 文件，T-Note 会自动打开

**方法三：最近打开**
- 从"文件" → "最近打开"中选择

### 保存笔记

- **自动保存**：编辑后 3 秒自动保存（需要先保存过一次）
- **手动保存**：按 Ctrl+S 或点击"保存"按钮
- **另存为**：点击"文件" → "另存为"，保存到新位置

### 修改笔记名称

1. 点击顶部的笔记名称
2. 输入新名称
3. 按回车确认

> 注意：修改笔记名称不会修改文件名，只修改笔记内部的名称。

## 页面管理

### 创建页面

1. 点击左侧"页面"图标
2. 点击"新建页面"按钮
3. 新页面会自动命名为"第N页"，可以在编辑器中修改标题

### 切换页面

- **方法一**：点击左侧页面列表中的页面
- **方法二**：点击 Tab 栏中的 Tab
- **方法三**：通过搜索功能找到页面

### 删除页面

1. 在页面列表中找到要删除的页面
2. 点击右侧的删除图标
3. 确认删除
4. 页面会进入回收站，可以恢复

### 页面信息

每个页面显示以下信息：
- **标题**：页面的标题
- **标签**：页面的标签（彩色显示）
- **更新时间**：最后修改时间

## 富文本编辑

T-Note 使用 Quill 2.0 编辑器，提供强大的富文本编辑功能。

### 工具栏功能

#### 文本格式
- **粗体**：Ctrl+B 或点击 B 按钮
- **斜体**：Ctrl+I 或点击 I 按钮
- **下划线**：Ctrl+U 或点击 U 按钮
- **删除线**：点击 S 按钮

#### 标题
- **H1**：一级标题
- **H2**：二级标题
- **H3**：三级标题

#### 列表
- **有序列表**：1. 2. 3. 格式
- **无序列表**：• 格式

#### 颜色
- **文字颜色**：点击 A 按钮选择颜色
- **背景色**：点击背景色按钮选择颜色

#### 其他
- **对齐方式**：左对齐、居中、右对齐
- **链接**：插入超链接
- **图片**：插入图片
- **代码块**：插入代码块
- **引用**：插入引用块

### 编辑技巧

#### 快速格式化
1. 选中文本
2. 点击工具栏按钮应用格式

#### 清除格式
1. 选中文本
2. 点击"清除格式"按钮

#### 插入图片
1. 点击图片按钮
2. 选择图片文件
3. 图片会嵌入到笔记中

#### 代码块
1. 点击代码块按钮
2. 输入代码
3. 支持语法高亮

## Tab 栏和分屏

T-Note 提供强大的 Tab 栏和分屏功能，让你可以同时查看和编辑多个页面。

### Tab 栏基础

#### 打开 Tab
- 从左侧页面列表点击页面，会自动添加到左侧 Tab 栏
- 最多可以打开 5 个 Tab

#### 切换 Tab
- 点击 Tab 栏中的任意 Tab 切换到对应页面
- 当前激活的 Tab 会高亮显示

#### 关闭 Tab
- 点击 Tab 右侧的 × 按钮关闭 Tab
- 关闭 Tab 不会删除页面，只是从 Tab 栏移除

#### Tab 排序
- 拖动 Tab 可以调整顺序
- 水平拖动进行排序

### 分屏功能

T-Note 支持左右分屏，可以同时编辑两个页面。

#### 开启分屏

**方法一：右键菜单**
1. 右键点击 Tab
2. 选择"分屏显示"
3. Tab 会移动到右侧

**方法二：拖拽到分屏区域**
1. 拖动 Tab 到右侧的"拖到这里分屏"区域
2. 松开鼠标，Tab 会移动到右侧

**方法三：向右拖动**
1. 拖动 Tab 向右超过 100px
2. 松开鼠标，自动分屏

#### 关闭分屏
- 点击右上角的"关闭分屏"按钮
- 或者关闭右侧所有 Tab

### 焦点切换

- 同一时间只有一个编辑器可以编辑
- 点击左侧或右侧编辑器切换焦点
- 当前焦点的编辑器会有边框高亮

### 分屏编辑技巧

1. **对比编辑**：左右分屏显示两个相关页面，方便对比和参考
2. **复制内容**：从一个页面复制内容到另一个页面
3. **独立滚动**：左右编辑器独立滚动，互不影响
4. **实时保存**：两个编辑器的修改都会实时保存

## 书签功能

书签可以帮助你快速定位到页面中的重要位置。

### 添加书签

1. 在编辑器中选中要添加书签的文本
2. 点击工具栏的"添加书签"按钮
3. 输入书签名称和备注（可选）
4. 点击"添加"

### 查看书签

1. 点击左侧"书签"图标
2. 会显示所有页面的书签列表
3. 书签按页面分组显示

### 跳转到书签

1. 在书签列表中找到要跳转的书签
2. 点击书签
3. 编辑器会自动跳转到书签位置并高亮显示

### 编辑书签

1. 在书签列表中找到要编辑的书签
2. 点击"编辑"按钮
3. 修改书签名称或备注
4. 点击"保存"

### 删除书签

1. 在书签列表中找到要删除的书签
2. 点击"删除"按钮
3. 确认删除
4. 书签会进入回收站，可以恢复

## 搜索功能

T-Note 提供强大的搜索功能，帮助你快速找到需要的内容。

### 按标签搜索

1. 点击左侧"搜索"图标
2. 在"按标签搜索"输入框中输入标签名
3. 会实时显示包含该标签的页面

### 按内容搜索

1. 点击左侧"搜索"图标
2. 在"搜索内容"输入框中输入关键词
3. 会显示包含该关键词的页面和位置

### 搜索结果

搜索结果会显示：
- **页面标题**
- **匹配的内容片段**（高亮显示关键词）
- **标签**
- **更新时间**

### 跳转到搜索结果

1. 点击搜索结果
2. 编辑器会自动打开对应页面
3. 并跳转到匹配的位置

## TODO 管理

T-Note 内置 TODO 管理功能，帮助你管理待办事项。

### 添加 TODO

1. 点击左侧"TODO"图标
2. 点击"添加待办"按钮
3. 填写以下信息：
   - **标题**（必填）
   - **描述**（可选）
   - **优先级**：低、中、高
   - **分类**（可选）
   - **截止日期**（可选）
4. 点击"添加"

### 查看 TODO

TODO 列表显示所有待办事项，包括：
- **标题和描述**
- **优先级标识**（颜色标记）
- **分类**
- **截止日期**
- **完成状态**

### 完成 TODO

- 点击 TODO 前面的复选框标记为完成
- 已完成的 TODO 会显示删除线
- 再次点击可以取消完成

### 编辑 TODO

1. 点击 TODO 右侧的"编辑"按钮
2. 修改信息
3. 点击"保存"

### 删除 TODO

1. 点击 TODO 右侧的"删除"按钮
2. 确认删除
3. TODO 会进入回收站，可以恢复

### 过滤 TODO

可以按以下条件过滤：
- **状态**：全部、未完成、已完成
- **优先级**：全部、高、中、低

### TODO 排序

TODO 会自动按以下规则排序：
1. 未完成的在前，已完成的在后
2. 优先级高的在前
3. 截止日期近的在前

## 云同步功能

T-Note 支持云同步，可以将笔记同步到云端，实现多设备访问。

### 支持的云盘

- **OneDrive**：微软云盘
- **百度网盘**：百度云盘

### 添加云盘账号

1. 点击左侧"设置"图标
2. 选择"云存储账号"
3. 点击"添加账号"
4. 选择云盘类型（OneDrive 或百度网盘）
5. 按照提示完成授权
6. 设置账号显示名称
7. 选择同步文件夹

### 管理云盘账号

在"云存储账号"面板中，可以：
- **查看账号信息**：显示名称、用户信息、同步文件夹
- **设置为默认账号**：用于快速上传
- **编辑账号**：修改显示名称、同步文件夹
- **断开连接**：断开账号连接
- **删除账号**：删除账号

### 同步设置

每个账号可以单独设置：
- **仅 WiFi 同步**：只在 WiFi 环境下同步
- **保存冲突副本**：发生冲突时保存副本

### 上传笔记到云端

**方法一：快速上传**
1. 点击顶部的云同步按钮
2. 选择"上传到云端"
3. 选择账号（如果有多个）
4. 笔记会上传到默认账号的同步文件夹

**方法二：从云笔记面板上传**
1. 点击左侧"云笔记"图标
2. 点击"上传当前笔记"
3. 选择账号和目标文件夹

### 浏览云端笔记

1. 点击左侧"云笔记"图标
2. 选择账号
3. 会显示云端笔记列表

### 预览云端笔记

1. 在云端笔记列表中找到笔记
2. 点击"预览"按钮
3. 笔记会在预览模式下打开（不保存到本地）
4. 顶部会显示"[预览]"标识

### 下载云端笔记

1. 在云端笔记列表中找到笔记
2. 点击"下载"按钮
3. 选择保存位置
4. 笔记会下载到本地

### 删除云端笔记

1. 在云端笔记列表中找到笔记
2. 点击"删除"按钮
3. 确认删除
4. 笔记会从云端删除

### 同步状态

笔记的同步状态会显示在页面列表中：
- **未同步**：灰色图标
- **已同步**：绿色图标
- **同步中**：旋转图标
- **冲突**：红色图标
- **错误**：黄色图标

### 解决同步冲突

当本地和云端都有修改时，会出现冲突：
1. 系统会弹出冲突解决对话框
2. 可以选择：
   - **使用本地版本**：覆盖云端版本
   - **使用云端版本**：覆盖本地版本
   - **保存两个版本**：保存冲突副本
3. 选择后点击"确定"

### 离线模式

当网络不可用时，T-Note 会自动进入离线模式：
- 顶部会显示"离线模式"标识
- 可以正常编辑笔记
- 修改会在网络恢复后自动同步

## 回收站

回收站保存已删除的项目，可以恢复或永久删除。

### 查看回收站

1. 点击左侧"回收站"图标
2. 会显示所有已删除的项目

### 回收站内容

回收站可以包含：
- **页面**：已删除的页面
- **书签**：已删除的书签
- **TODO**：已删除的待办事项

### 恢复项目

1. 在回收站中找到要恢复的项目
2. 点击"恢复"按钮
3. 项目会恢复到原来的位置

### 永久删除

1. 在回收站中找到要永久删除的项目
2. 点击"永久删除"按钮
3. 确认删除
4. 项目会被永久删除，无法恢复

### 清空回收站

1. 点击"清空回收站"按钮
2. 确认操作
3. 所有项目会被永久删除

## 常见问题

### 1. 如何备份笔记？

笔记以 .note 文件格式保存在本地，可以直接复制文件进行备份。建议：
- 定期复制 .note 文件到其他位置
- 使用云同步功能自动备份到云端
- 使用系统的文件备份功能

### 2. 笔记文件可以在不同电脑上使用吗？

可以。.note 文件是跨平台的，可以在不同电脑上使用。方法：
- 通过 U 盘或网盘传输文件
- 使用云同步功能自动同步

### 3. 如何导出笔记？

目前 T-Note 使用 .note 格式保存，可以：
- 直接复制 .note 文件
- 从编辑器中复制内容到其他应用
- 未来版本会支持导出为 PDF、Markdown 等格式

### 4. 笔记文件很大怎么办？

如果笔记文件过大，建议：
- 将内容拆分到多个笔记文件
- 删除不需要的页面
- 清空回收站
- 压缩或删除大图片

### 5. 云同步失败怎么办？

如果云同步失败，可以尝试：
- 检查网络连接
- 重新授权云盘账号
- 检查云盘空间是否充足
- 查看错误日志
- 手动上传或下载

### 6. 如何恢复误删的内容？

- **页面、书签、TODO**：从回收站恢复
- **文本内容**：使用 Ctrl+Z 撤销
- **整个笔记文件**：从备份或云端恢复

### 7. 分屏白屏怎么办？

如果分屏出现白屏：
- 关闭分屏重新打开
- 重启应用
- 清除缓存
- 检查控制台错误

### 8. 如何更新 T-Note？

T-Note 支持自动更新：
- 有新版本时会在顶部显示提示
- 点击"下载更新"按钮
- 下载完成后点击"安装更新"
- 应用会自动重启并安装更新

### 9. 如何卸载 T-Note？

- Windows：在"控制面板" → "程序和功能"中卸载
- 卸载后，笔记文件不会被删除

### 10. 如何联系支持？

- GitHub Issues: [提交问题](https://github.com/DBAA-LCT/T-NOTE/issues)
- 查看更新日志了解最新功能和修复
