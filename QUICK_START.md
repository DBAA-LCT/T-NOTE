# 快速启动指南

## 🚀 立即开始

### 1. 确认依赖已安装
项目已经包含了 Ant Design 6.0 的依赖，如果是首次运行，请先安装：

```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

这会同时启动：
- Vite 开发服务器（前端）
- Electron 应用（桌面窗口）

### 3. 构建生产版本
```bash
npm run build
```

### 4. 打包为 Windows 应用
```bash
npm run package
```

## 📝 使用提示

1. **首次使用**
   - 点击"新建页面"创建第一个页面
   - 输入标题和内容
   - 使用 Ctrl+S 或点击"保存"按钮保存

2. **标签管理**
   - 在标签输入框输入标签名
   - 按 Enter 或点击"添加"按钮
   - 点击标签上的 × 删除标签
   - 使用搜索框按标签筛选页面

3. **文件操作**
   - "打开"：打开已保存的 .json 笔记文件
   - "保存"：保存到当前文件（首次会提示选择位置）
   - "另存为"：保存到新位置

4. **编辑技巧**
   - 使用工具栏格式化文本
   - 鼠标悬停在工具栏按钮上查看中文提示
   - 代码块适合记录代码片段
   - 使用颜色标记重点内容

## ⚠️ 注意事项

- 确保 Node.js 版本 >= 16
- 首次运行需要安装依赖
- 笔记文件为 JSON 格式
- 建议定期备份重要笔记
- Ctrl+S 快捷键保存

## 🐛 常见问题

**Q: 样式显示不正常？**
A: 清除缓存后重新运行 `npm run dev`

**Q: 保存失败？**
A: 检查文件路径是否有写入权限

**Q: 图标不显示？**
A: 确认 @ant-design/icons 已正确安装

**Q: Electron 窗口不显示？**
A: 检查 electron 进程是否正常启动
